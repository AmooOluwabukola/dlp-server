
#  Auth API

A scalable backend application built to power modern web applications with secure authentication, structured APIs, and reliable business logic.

##  Features

- RESTful API architecture
- Authentication 
- Database management
- Input validation and error handling
- Modular and scalable structure

## 🛠️ Tech Stack
- Node.js
- NestJS 
- TypeScript
- MongoDB 
- Mongoose 
- JWT Authentication


##  Installation

Clone the repository
https://github.com/AmooOluwabukola/dlp-server.git

Navigate into the project directory:
cd dlp

Install dependencies:
npm install

Start the development server:
npm run start:dev

##  Environment Variables
Create a .env file in the root directory:
PORT=
DATABASE_URL=
JWT_SECRET=
